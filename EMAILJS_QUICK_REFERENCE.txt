ServiceID:    Dashboard → Email Services → [Your Service]
TemplateID:   Dashboard → Email Templates → [Your Template]  
PublicKey:    Dashboard → Account → API Keys

⚠️  Important Notes:
• Use Public Key (not Private Key) in client-side code
• Service ID starts with "service_"
• Template ID starts with "template_"
• Public Key is alphanumeric string
